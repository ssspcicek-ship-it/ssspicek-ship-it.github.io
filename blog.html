What is an HTTP request?

When I type a web address into a browser, my computer sends an HTTP request to a server. This request asks for a resource such as a web page, an image, or a file.

The server then sends back an HTTP response with a status code and the requested content. My browser reads the response and displays the result on the screen.

Registers and main memory

Registers are very small and very fast storage locations inside the CPU. They hold data that the processor is using immediately.

Main memory, or RAM, is much larger but slower. Programs use both because fast access matters, but large storage also matters.

Normalisation in databases

Normalisation is the process of organising tables so that data is stored more clearly and with less duplication.

This helps improve consistency and makes updates easier. It also reduces the chance of errors when data changes.
