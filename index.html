Production diary – Week 15

Date: 13 February 2026

This week I developed my GitHub Pages portfolio and improved the structure into clear sections (About, Projects, Skills, Contact). I also started a CS Maths & Tools page and implemented interactive JavaScript tools.

The achievement I am most proud of is the Maths & Tools page because it demonstrates real concepts such as binary representation and exponential growth, not just layout.

The most challenging part was organising HTML structure clearly and debugging JavaScript when user input was invalid. Writing concise professional text in UK English also required careful wording.

I learned how binary maps to powers of two and why it fits hardware, and how exponential growth explains why password length increases security so quickly.

Next week I want to add one more project with a clearer technical write-up and improve accessibility (navigation clarity and keyboard focus).
