What is an HTTP request?

When I type a website address into my browser and press Enter, my computer sends an HTTP request to a web server. This request asks for a specific resource, such as a page, an image, or a stylesheet. The server then sends back an HTTP response with a status code and the content the browser should display. My browser reads the response and shows the page on screen. This process happens many times on one website because pages often load several files, not just one.

6 March 2026

Registers vs main memory

CPU registers are very small and very fast storage locations inside the processor. They hold data that the CPU is currently using. Main memory, or RAM, is much larger but slower. Programs rely on both: registers for immediate work and RAM for storing larger amounts of active data. Understanding this helps explain why data movement and memory access matter for performance.

6 March 2026

Intro to normalisation in databases

Normalisation is a way of organising database tables so that data is stored clearly and with less duplication. Instead of repeating the same information in many places, related data is split into sensible tables. This improves consistency and makes updates easier. It also reduces the risk of errors when the database grows.
