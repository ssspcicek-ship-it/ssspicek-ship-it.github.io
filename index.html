Production diary – Week 15

Date: 13 February 2026

This week I developed my GitHub Pages portfolio and improved the structure into clear sections (About, Projects, Skills, Contact). I also started a dedicated CS Maths & Tools page and implemented interactive JavaScript tools.

The achievement I am most proud of is the Maths & Tools page. It demonstrates real computer science ideas (binary representation and exponential growth) rather than only visual design.

The most challenging part was organising the HTML structure clearly and debugging small JavaScript issues when user input was invalid. Writing concise, professional text in UK English also required careful wording.

I learned more about how binary uses powers of two and why that maps well to hardware. I also learned how exponential growth explains password security, because increasing length increases combinations extremely fast.

Next week I want to add one stronger project with a clear technical write-up and improve the site’s accessibility (better contrast and clearer focus states for keyboard navigation).
