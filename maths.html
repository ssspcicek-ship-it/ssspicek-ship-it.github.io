CS Maths & Tools

This page contains small interactive tools based on simple computer science maths.

Binary and decimal converter

This tool converts a decimal number into binary and hexadecimal.

In binary, each position represents a power of 2. This is useful in computing because digital hardware naturally works with two states.

2D distance calculator

This tool calculates the straight-line distance between two points.

This formula is used in graphics, games, maps, and pathfinding.

Password combinations estimator

This tool estimates how many possible passwords exist based on password length and character set size.

If each character can be chosen in N ways and the password has L characters, the total number of combinations is N to the power of L.
