Computer Science Maths & Tools

This page demonstrates practical computer science maths concepts using JavaScript.

Binary Converter

Convert decimal numbers to binary and hexadecimal.

Binary uses base 2. Each digit represents a power of two (1, 2, 4, 8, 16, ...). This is used in computer hardware because circuits operate in two states: on and off.

Password Combinations Estimator

If a password has N possible characters and length L, the number of combinations is N^L.

Exponential growth means that increasing password length dramatically increases total combinations.
